# ドキュメント

## 目的

- **バージョン管理システムである Git と共有プラットフォームである GitHub の使い方を習得**

  - Git の基本操作（clone、add、commit、push、pull、branch、merge など）の習得
  - GitHub を活用した、プルリクエストベースの開発フロー（GitHub Flow）の理解と実践
  - コンフリクト（競合）の解決方法の理解と実践
